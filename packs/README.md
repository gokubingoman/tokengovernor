# TokenGovernor **`packs/`**

Extended rules and paste workflows that complement **`universal/`** and **`cursor/`**.

| Path | Purpose |
|------|---------|
| `claude/`, `gemini/`, `opencode/`, `openclaw/` | Tool-specific rule files |
| `commands/` | Session prompts (`tg-compact`, `tg-low-token`, …) |
| `examples/` | Before/after discipline |
| `memory/` | Retrieval-first policy; **tiered context loading** |
| `security/` | Approval-gate checklist |

After you copy this repo into your project, this folder usually lives at **`tokengovernor/packs/`** next to `universal/` and `cursor/`. Paths inside files assume that layout (adjust if you copy subtrees only).

**More examples and industry packs** ship with **TokenGovernor Plus**.
