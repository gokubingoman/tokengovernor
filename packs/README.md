# TokenGovernor **`packs/`**

Tool-specific rules, paste-ready commands, and expanded policy that sit alongside **`universal/`** and **`cursor/`**.

| Path | Contents |
|------|----------|
| **`claude/`**, **`gemini/`**, **`opencode/`**, **`openclaw/`** | Agent- or IDE-specific rule files |
| **`commands/`** | Session prompts (`tg-compact`, `tg-low-token`, …) |
| **`examples/`** | Before / after discipline illustration |
| **`memory/`** | Retrieval-first policy; tiered context loading |
| **`security/`** | Approval-gate checklist |

When vendored, this directory usually lives at **`tokengovernor/packs/`**. Paths inside the Markdown assume that layout — adjust references if you relocate subtrees.

**Additional examples and industry packs** ship with **TokenGovernor Plus**.
