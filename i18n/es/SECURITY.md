# Seguridad

TokenGovernor Lite es **documentación y reglas** (Markdown). No es un servicio en tu infraestructura. Aun así, instrucciones maliciosas en archivos de confianza podrían afectar al agente.

## Informar una vulnerabilidad

- Usa el [**informe privado de seguridad de GitHub**](https://github.com/gokubingoman/tokengovernor/security/advisories/new) si crees haber encontrado un problema de seguridad.
- **No** abras issues públicos para vulnerabilidades sin divulgar.

Para fallos **no sensibles**, usa [GitHub Issues](https://github.com/gokubingoman/tokengovernor/issues).

## Alcance

Este repositorio no debe contener secretos en vivo, claves API ni datos de clientes. Si detectas una filtración accidental, infórmalo por un canal privado de inmediato.
