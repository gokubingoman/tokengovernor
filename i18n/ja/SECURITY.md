# セキュリティ

TokenGovernor Lite は**ドキュメントとルール**（Markdown）です。インフラ上のサービスとしては動きません。ただし、信頼するファイルに悪意のある指示が混ざればエージェント挙動に影響し得ます。

## 脆弱性の報告

- セキュリティ上の問題と思われる場合は、GitHub の [**プライベートなセキュリティ報告**](https://github.com/gokubingoman/tokengovernor/security/advisories/new) を使用してください。
- 未公開の脆弱性を**公開 issue に書かない**でください。

**機密性の低い**不具合や改善は [GitHub Issues](https://github.com/gokubingoman/tokengovernor/issues) でどうぞ。

## 範囲

本リポジトリに稼働中の秘密、API キー、顧客データを置かないでください。誤って公開した場合は、直ちに非公開の連絡手段で報告してください。
