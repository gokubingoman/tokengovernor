# `packs/` 概要（人向け）

エージェントが実際に読む英語ルールは [`packs/`](../../packs/)。本ページは**日本語**で各パーツの役割を説明します。

| パス（英語） | 役割 |
|--------------|------|
| **`packs/claude/CLAUDE.md`** | Claude Code 向け。 |
| **`packs/gemini/GEMINI.md`** | Gemini CLI 向け。 |
| **`packs/opencode/AGENTS.md`** | OpenCode / エージェント向けシステム追記。 |
| **`packs/openclaw/SKILL.md`** | OpenClaw / claw 系ワークフロー。 |
| **`packs/commands/`** | 貼り付け用プロンプト（要約、低トークンモードなど）。 |
| **`packs/examples/before-after.md`** | 規律の before/after 例。 |
| **`packs/memory/`** | メモリ方針と階層読み込み。 |
| **`packs/security/approval-gates.md`** | 人の承認が必要な操作の一覧。 |

**Plus** では追加の packs がある場合があります：[tokengovernor-plus](https://github.com/gokubingoman/tokengovernor-plus)。
